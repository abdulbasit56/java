Battleship Game - User Manual

1. Introduction
Welcome to the Battleship game! Your objective is to locate and destroy all enemy ships hidden within the grid.

2. How to Play
- Enter your name when prompted.
- The game will display a leaderboard with the top scores.
- To start the game, press "1" when prompted; otherwise, exit.
- The game consists of three levels with increasing difficulty.

3. Game Levels
- Level 1: 4x4 grid with 1 ship. You have 5 tries to sink it.
- Level 2: 5x5 grid with 2 ships. You have 10 tries.
- Level 3: 6x6 grid with 3 ships. You have 12 tries.

4. Attacking
- Enter the row and column number (1-based) where you want to attack.
- If you hit a ship, it will be marked.
- If you miss, you lose a try.
- If you run out of tries, you lose the level.

5. Hints
- You can request hints by sacrificing a try.
- Hints may reveal the row or column of the ship using a riddle.

6. Winning & Leaderboard
- Successfully hitting all ships in a level moves you to the next level.
- Your final score is recorded in the leaderboard if you exit or lose.
- The leaderboard shows the top 10 highest scores.

7. Exiting the Game
- If you choose not to continue at any stage, the game will exit and save your score.
- You can restart by launching the game again.

8. Additional Features
- Animated text display for better user experience.
- Randomized ship placements for unique gameplay each time.

**Note:** This project was developed as part of Java programming fundamentals and follows a basic implementation with less emphasis on efficiency. By applying an object-oriented approach, the code structure could be significantly improved, making it more modular, maintainable, and efficient.

Enjoy the game and aim for the top of the leaderboard!
